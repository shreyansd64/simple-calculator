# Simple Calculator 

A basic calculator program written in Python for my CSE Project Management course.

## Description
This is a simple command-line calculator that can perform basic arithmetic operations.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- Error handling for division by zero
- User-friendly interface

## How to Use
1. Run the program
2. Enter your first number
3. Enter your second number
4. Choose an operation (+, -, *, /)
5. Get your result!

## Example
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): +
✓ Result: 10 + 5 = 15
